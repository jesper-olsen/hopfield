Hopfield Networks
=================

Train classic Hopfield networks [1,2,3] to store memories (bit vectors) using Hopfield's one-shot storage rule or the iterative perceptron convergence procedure.

Two examples included:
* [Font8x8](README_font8x8.md) - small number of static memories (8x8 bitmaps).
* [MNIST](README_mnist.md) - MNIST handwritten digit classification.
